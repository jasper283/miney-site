## Build output

The project uses Next.js static export:

```mjs
const nextConfig = {
  output: 'export',
  trailingSlash: true,
};
```

`npm run build` writes the production site to `out`.

## Pages settings

Use these Cloudflare Pages settings:

- Framework preset: `Next.js (Static HTML Export)`
- Build command: `npx next build`
- Build output directory: `out`
- Node version: `22`

## Static-only policy

Keep the site deployable as static assets. If Miney later needs forms, auth, server-rendered pages, or
API endpoints, choose that explicitly and move to a Cloudflare Workers/OpenNext deployment model.
